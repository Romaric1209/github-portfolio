GAIDI - GenAI Data Identifier

Have you ever wondered if what you were looking at was real? If what you were reading had been written by a person or not? I did, especially in the age we are now living in, where misinformation and fake news are normal.

F. Romaric Berger

Creator

I have created an app that analyse Texts and Images. It then classifies it between two categories, generated with Artificial Intelligence, or not.

üìù Text Prediction

The text model has been created with datasets from Kaggle:

  • Human vs LLM Text Corpus

    Working on the whole dataset being impossible considering the size of it and the material available (my old personal laptop), I extracted 13,000 rows from this dataset, ensuring that the source is 50% human and 50% AI-generated from models released after 2023.

  • AI vs Human Text Dataset

    This dataset gave me more content to train on.

With these two datasets, I trained a pipeline on over 63,000 rows.

1. Text Cleaning & Preprocessing - Pipeline
  • Lowercases the entire text
  • Removes special characters or excessive whitespace
  • Tokenizes using advanced models like spaCy
2. Feature Extraction - Pipeline
  • TF-IDF: Measures word importance in a larger corpus (max_features:27000, ngram_range:(1,2))
  • Readability: Uses scores like Flesch Reading Ease, Gunning Fog
  • Sentiment: Analyzes tone (positive/negative/neutral) using VADER
  • Redundancy: Looks for repetitive patterns common in AI text
3. Prediction Using Machine Learning

These features are fed into a trained SVM (Support Vector Machine) model that predicts:

  • ‚úÖ Human-written
  • ü§ñ AI-generated

GAIDI can also analyze images using a deep learning model:

I used these datasets from Kaggle:

  • CIFAKE Dataset

    It has many pictures, but focused on the ten categories of the original Cifar dataset.

  • Shutterstock AI vs Human

    This dataset provides landscape and many random items to the training.

  • RVF10K Dataset

    This dataset provides material for the DNN to train on human faces.

The model :

  • Resizes and normalizes the image
  • Uses a neural network trained to classify AI vs Human images
